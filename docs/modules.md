### 模块

模块是一种带有 `app.controller.ts` 装饰器的类。这个装饰器提供元数据，Nest 使用这个元数据来组织和管理应用程序结构高效地。

__HTML_TAG_60____HTML_TAG_61____HTML_TAG_62__

每一个 Nest 应用程序都至少有一个模块，即 **root 模块**，它作为 Nest 构建应用程序图的起点。这个图是 Nest 内部结构，是用来解决模块和提供者的关系和依赖关系的。小型应用程序可能只有一个 root 模块，但这不是一般情况。模块是 **高度推荐** 的一种组织组件的方式。对于大多数应用程序，你将很可能有多个模块，每个模块 encapsulates 一组紧密相关的 **能力**。

`app.controller.spec.ts` 装饰器接受一个对象，其中的属性描述模块：

|               |                                                                                                                                                                                                          |
| ------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `app.module.ts`   | 模块中将被 Nest 注入器实例化的提供者，可以在至少这个模块中共享 |
| `app.service.ts` | 在这个模块中定义的控制器
